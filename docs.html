📚 NPF - Documentación Completa

🚀 Bienvenido a NPF

NPF es una aplicación web de gestión financiera personal construida con React. Permite gestionar tus cuentas, transacciones, categorías y visualizar reportes.

📋 Contenido Principal:

  • Gestión de cuentas bancarias y monederos
  • Categorización de transacciones
  • Reportes y gráficos
  • Autenticación de usuarios
  • Tema claro/oscuro

🛠️ Tecnologías

  • React 18.2 - Librería UI
  • React Router 6 - Navegación
  • Tailwind CSS 3 - Estilos
  • Axios - Cliente HTTP
  • LocalForage - Almacenamiento local

🚀 Tutorial para Principiantes

Paso 1: Entender el Proyecto

NPF es una app de finanzas personales. Permite:

  • Registrarse y autenticarse
  • Crear cuentas bancarias
  • Registrar movimientos
  • Ver reportes

Paso 2: Instalación

# 1. Navega al proyecto
cd C:\laragon\www\npf

# 2. Instala dependencias
npm install

# 3. Inicia desarrollo
npm start

Paso 3: Hacer tu Primer Cambio

Abre src/components/buttons/Button.jsx y cambia:

// De:
className={`px-4 py-2 rounded bg-blue-500 text-white ${className}`}

// A:
className={`px-4 py-2 rounded bg-green-500 text-white ${className}`}

¡Los botones cambiarán a verde automáticamente!

Paso 4: Crear un Componente

Crea archivo: src/components/cards/MyCard.jsx

export function MyCard({ title, value }) {
  return (
    <div className="p-4 rounded-lg bg-white shadow-md">
      <h3 className="text-lg font-bold">{title}</h3>
      <p className="text-2xl font-bold text-green-600">${value}</p>
    </div>
  );
}

Úsalo en una página importando: import { MyCard } from '../components/cards/MyCard';

👶 Primeros Pasos de Usuario

Registro e Login

  1. Ve a http://localhost:3000
  2. Haz clic en "Registrarse"
  3. Completa el formulario
  4. Se guardará tu sesión automáticamente

Crear tu Primera Cuenta

  1. Navega a "Cuentas" en el menú
  2. Haz clic en "+"
  3. Llena el formulario (nombre, categoría, saldo inicial)
  4. Haz clic en "Guardar"

Registrar tu Primer Movimiento

  1. Navega a "Registros"
  2. Haz clic en "+ Agregar"
  3. Selecciona origen y destino
  4. Ingresa monto y descripción
  5. Haz clic en "Guardar"

🔌 Servicios API

auth-services.js

Maneja autenticación:

import { authService } from '../services/auth-services';

// Login
const resultado = await authService.login(email, password);

// Registro
const nuevoUsuario = await authService.register(datos);

// Logout
await authService.logout();

data-services.js

Gestiona operaciones CRUD:

import { dataService } from '../services/data-services';

// Obtener
const datos = await dataService.getData(url, id, filters, token);

// Crear
const nuevo = await dataService.createData(url, datos, token);

// Actualizar
const actualizado = await dataService.updateData(url, id, datos, token);

// Eliminar
await dataService.deleteData(url, id, token);

│   API Backend (REST)               │
└────────────────────────────────────┘

Flujo de Autenticación

  1. Usuario ingresa credenciales
  2. Se envía POST a /login
  3. API devuelve token JWT
  4. Se guarda en localStorage con localforage
  5. Se envía en header de peticiones: Authorization: Bearer token

Patrones de Diseño

  • Service Pattern - Servicios centralizados para lógica
  • Component Composition - Componentes reutilizables
  • Custom Hooks - Lógica reutilizable (usePagination)
  • Layout/Wrapper - Componentes contenedores

❓ Problemas Comunes

❌ "npm: command not found"

Solución: Instala Node.js desde nodejs.org

❌ "Port 3000 already in use"

Solución: Ejecuta en otro puerto:

npm start -- --port 3001

❌ Los estilos no se aplican

Solución: Reinicia el servidor:

npm start

❌ "Cannot find module"

Solución: Reinstala dependencias:

npm install

❌ API no responde

Solución: Verifica:

  • URL correcta en src/config/main.config.js
  • Backend esté ejecutándose
  • No haya problemas CORS

❌ No puedo hacer login

Causa posible: Credenciales incorrectas o backend no responde

Solución: Verifica token en DevTools (F12) → Application → LocalStorage

✅ Si nada funciona

# 1. Borra node_modules
rm -r node_modules

# 2. Reinstala
npm install

# 3. Reinicia
npm start
